## Setup System
To run this project:

```
$ git clone git@github.com:bansalsumit/areferralguru.git
$ cd ./string_calculator_tdd
$ rvm install 2.7.5
$ docker build -t string_calculator_tdd
$ docker run string_calculator_tdd
```
